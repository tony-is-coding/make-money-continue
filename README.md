# Make Money Continue — 认知引擎知识库

每日四维研报 + 交叉洞察，持续积累认知资产。

## 目录结构

```
daily/YYYYMMDD/
├── README.md         — 当期汇总（核心论点 + 交叉洞察 + 行动建议）
├── finance.md        — 财经深度分析
├── ai-tech.md        — AI技术深度分析
├── ai-industry.md    — AI产业架构与趋势
└── product.md        — 产品拆解深度分析

weekly/YYYYMMDD/
├── README.md         — 周度汇总
└── analysis.md       — 周度深度综合

monthly/YYYYMM/
├── README.md         — 月度汇总
└── analysis.md       — 月度深度综合

insights/
├── themes.md         — 持续追踪的主题线索
└── predictions.md    — 预测记录（事后验证）
```

## 产出节奏

| 时间 | 产出 |
|------|------|
| 每天 09:00 | daily/YYYYMMDD/ 四份深度研报 + README 汇总 |
| 每周日 20:00 | weekly/YYYYMMDD/ 周度综合 |
| 每月末 | monthly/YYYYMM/ 月度综合 |

## 质量标准

- 每篇深度分析 1500-3000 字
- 必须有核心论点（一句话）
- 必须有证据链（数据/事实/来源）
- 必须有 So What（行动建议）
- 必须有交叉关联（跨领域连接）
- 每期 README.md 是入口，提供快速概览

## 分支策略

单分支 `main`，所有内容直接 push。简单即正义。
