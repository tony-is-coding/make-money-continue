# Make Money Continue — 认知引擎知识库

每日四维研报 + 交叉洞察，持续积累认知资产。

## 目录结构

```
daily/YYYY-MM-DD/
├── finance.md        — 财经研报
├── ai-tech.md        — AI技术研报
├── ai-industry.md    — AI产业架构与趋势
├── product.md        — 产品拆解
└── digest.md         — 交叉洞察摘要（主编产出）

weekly/
└── YYYY-WNN.md       — 周度综合分析

insights/
├── themes.md         — 持续追踪的主题线索
└── predictions.md    — 预测记录（事后验证）

templates/            — 报告模板
```

## 产出节奏

| 时间 | 产出 |
|------|------|
| 每天 07:00 | 选题确定 |
| 每天 09:00 | 四份研报 + 交叉洞察 |
| 每周日 20:00 | 周度综合分析 |

## 质量标准

- 每篇必须有核心论点（一句话）
- 必须有证据链（数据/事实/来源）
- 必须有 So What（行动建议）
- 必须有交叉关联（跨领域连接）
- 控制在 2000 字以内
